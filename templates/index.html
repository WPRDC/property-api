property-api/single/

Searches WPRDC resources for records relating to a requested property or parcel.

Parameters

  • parcel_id - [text] Parcel identification number for property being searched

Response

  • success - [boolean] False if there were errors; True otherwise
  • help - [text] Helpful information regarding response
  • failed_searches - [list[text]] list of all WPRDC datasets where a search for parcel_id returned an 0 results
  • geo - {"latitude": [float], "longitude", [float]} coordinates of property centroid
  • results - [dict {dataset:list[results]}] JSON object mapping dataset to the data it contains relating to the queried parcel_id
